Before / After

記事改善before/after比較シート

内容サンプル

記事を改善したあと、Search Consoleの数値と変更内容を並べて、改善が効いたか確認するための比較シートです。スターターキットに含める予定のテンプレートの一部を、サンプルとして公開しています。

外部リンク、購入ボタン、メール登録はありません。改善前後の記録を残し、次の修正を落ち着いて判断するための手元用テンプレートです。

判定分類

  • 改善: clicksやCTRが上がり、avg positionも大きく悪化していない場合に選ぶ。
  • 横ばい: 数値が大きく変わらず、改善の影響をまだ判断しにくい場合に選ぶ。
  • 悪化: clicks、CTR、avg positionの複数が悪くなった場合に選ぶ。変更の戻しや再調整を検討します。
  • データ不足: impressionsが少ない、または改善後の期間が短く比較に足りない場合に選ぶ。

対象URL

何を見るか: 改善した記事や固定ページのURL。

どう判断するか: URLを変えずに、改善前後で同じページとして比較できるかを確認します。

記録欄:

改善日

何を見るか: title、本文、CTAなどを変更した日。

どう判断するか: Search Consoleの反映には時間差があるため、比較期間を分ける基準にします。

記録欄:

改善前title

何を見るか: 改善前に検索結果へ出ていたタイトル。

どう判断するか: 検索意図に答えているか、抽象的すぎないかを後から見直します。

記録欄:

改善後title

何を見るか: 改善後に差し替えたタイトル。

どう判断するか: 対象クエリとの自然な一致、クリックしたくなる具体性、誇大表現の有無を見ます。

記録欄:

改善前meta description

何を見るか: 改善前の説明文。

どう判断するか: 記事で何がわかるか、誰向けか、次に何ができるかが伝わっていたかを見ます。

記録欄:

改善後meta description

何を見るか: 改善後の説明文。

どう判断するか: 検索結果で本文内容とずれず、クリック前の期待値を整えられているかを見ます。

記録欄:

改善前impressions / clicks / CTR / avg position

何を見るか: 改善前期間のSearch Console指標。

どう判断するか: 表示はあるがクリックが弱いのか、順位が弱いのか、改善前の課題を切り分けます。

記録欄:

改善後impressions / clicks / CTR / avg position

何を見るか: 改善後期間のSearch Console指標。

どう判断するか: 同じURLで、改善前と比べて表示、クリック、CTR、平均掲載順位がどう動いたかを見ます。

記録欄:

変化の判定

何を見るか: 改善、横ばい、悪化、データ不足の分類。

どう判断するか: 1つの数値だけで決めず、impressions、clicks、CTR、avg positionをまとめて判断します。

記録欄:

次の改善方針

何を見るか: 次に直す場所や、まだ触らない判断。

どう判断するか: title再調整、本文追記、CTA見直し、またはデータが増えるまで待つ方針を記録します。

記録欄:

この比較シートは改善結果を確認するためのものです。検索順位、クリック数、収益を保証するものではありません。
